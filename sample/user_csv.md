# PCA Hub ユーザー CSV ファイルフォーマット

ファイル形式：<br/>
CSV（.csv）<br/>
<br/>

文字コード：<br/>
UTF-8（BOM付き）<br/>
<br/>

改行コード：<br/>
CRLF または CR または LF<br/>
<br/>

1行目：<br/>
バージョン情報（必須）<br/>
例 ）```Ver.1.0```<br/>
<br/>

2行目：<br/>
ヘッダー（必須）<br/>
```ユーザーID,ログイン名,メールアドレス,表示名,姓,名,姓カナ,名カナ,社員番号,ライセンス,所属グループ```<br/>
<br/>

3行目以降：<br/>
下表のとおりです。<br/>

|項目No.|項目名|上限値|必須|更新|制限|備考|
|--|--|--|--|--|--|--|
|1|ユーザーID|12文字|※|不可|半角英数字のみ|新規登録の場合には不要です。<br/>更新する場合には更新対象のユーザーIDを設定してください。<br/>大文字と小文字を区別しません。|
|2|ログイン名|128文字|○|不可|半角英数字 または 下記の記号のみ<br/>・ハイフン（-）<br/>・ピリオド（.）<br/>・アンダーバー（_）<br/>・アットマーク（@）|大文字と小文字を区別しません。|
|3|メールアドレス|128文字|○|不可|アットマーク（@）より前に英数字または下記の記号を1文字以上<br/>・ピリオド（.）<br/>・アンダーバー（_）<br/>・パーセント（%）<br/>・プラス（+）<br/>・ハイフン（-）<br/><br/>アットマーク（@）より後に英数字または記号 + ドット（.） + 英字1文字以上<br/>・ピリオド（.）<br/>・ハイフン（-）|大文字と小文字を区別しません。|
|4|表示名|160文字|○|不可|下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"）||
|5|姓|20文字|○|可|下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"）||
|6|名|20文字||可|下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"）||
|7|姓カナ|20文字|○|可|下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"）||
|8|名カナ|20文字||可|下記の記号を禁止<br/>・コロン（:）<br/>・ダブルクォーテーション（"）||
|9|社員番号|13桁||可|半角英数字のみ||
|10|ライセンス|| |不可||◆PCA Hub のサービス名を指定することでライセンスの付与を行うことができます。<br/><br/>◆改行区切りで複数のサービスを指定することができます。<br/><br/>◆ライセンスの解除を行うことはできません。<br/><br/><サービス名一覧><br/>・PCA Hub eDOC<br/>・PCA Hub 給与明細<br/>・PCA Hub 取引明細<br/>・PCA Hub 年末調整|
